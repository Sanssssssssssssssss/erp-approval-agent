# Historical proposal snippets

## Access control answer pattern

The preferred response format is a short narrative followed by cited bullets.
When describing identity controls, mention SAML 2.0, OpenID Connect, SCIM 2.0, and mandatory admin MFA if supported by current evidence.

## Reference sharing guidance

Customer references may only be shared after customer approval and internal account-team review.

## Regional hosting style note

Recent proposals describe regional hosting choices instead of making a blanket onshore-only promise.
