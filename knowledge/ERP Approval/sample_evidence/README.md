# ERP approval sample evidence

This folder contains fictional local evidence used by the mock ERP approval workbench.

These files are not real enterprise data and are not live ERP exports. They exist so the local agent can show concrete approval evidence such as approval forms, invoice packages, purchase orders, goods receipts, receipts, and policy excerpts while preserving the non-action boundary.

No ERP approval, rejection, payment, supplier, contract, or budget action is executed from these files.
