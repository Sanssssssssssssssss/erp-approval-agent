# ERP Approval Knowledge Placeholder

This folder is reserved for future ERP Approval Agent Workbench knowledge assets.

Future phases can add mock and local-first material such as:

- purchase requisition policies
- expense approval policies
- invoice and payment review checklists
- supplier onboarding requirements
- contract exception guidance
- budget exception policies
- mock ERP records for local development

Phase 0 does not add real ERP connectors or production policy content. The legacy [knowledge/RFP Security](../RFP%20Security) corpus remains in place for existing compatibility tests and benchmarks.
